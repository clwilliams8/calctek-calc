# Product

This section contains product specifications, user stories, and acceptance criteria for the CalcTek Calculator.

## Overview

CalcTek Calculator is a web-based, API-driven calculator that provides:

- **Basic arithmetic** -- addition, subtraction, multiplication, and division via a familiar keypad interface.
- **Expression evaluation** -- complex math expressions including `sqrt`, exponents (`^`), parentheses, and more, powered by MathExecutor on the backend.
- **Calculation history (Ticker Tape)** -- a scrollable log of every calculation the user has performed, persisted to the database.
- **Google Sign-In** -- secure authentication so each user's history is private and accessible across devices.
- **History management** -- ability to delete individual calculations or clear the entire history.
